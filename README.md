# 🍔 Swiggy Sales Analysis

A data analysis project focused on exploring and understanding sales performance from Swiggy food delivery data. This project uncovers insights related to customer behavior, order trends, revenue patterns, and restaurant performance using Python and data visualization techniques.

---

## 📌 Project Overview

Swiggy operates in a fast-paced food delivery ecosystem where understanding order patterns and customer preferences is crucial.

This project analyzes Swiggy sales data to:

- Identify peak order times
- Analyze revenue trends
- Understand customer purchasing behavior
- Evaluate restaurant or category performance
- Generate actionable business insights
